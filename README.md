🎵 Sal Priadi Fan Page
A responsive fan page website inspired by Sal Priadi, built using semantic HTML5 and CSS Flexbox layout.  
This project was created as a submission for Dicoding's Web Development course.

🌐 Live Demo
https://nazwaauliap.github.io/SalPriadi-Website-FansPage/
---

📌 Features
- ✅ Full-width hero section
- ✅ About section
- ✅ Albums section with embedded Spotify/YouTube content
- ✅ Sidebar profile with image
- ✅ Responsive layout using Flexbox
- ✅ Semantic HTML5 structure
- ✅ Clean and organized folder structure

---

🛠️ Built With
- HTML5
- CSS3 (Flexbox)
- Responsive Design (Media Queries)

🎯 Submission Criteria
This project fulfills the following Dicoding requirements:
- Semantic elements: `<header>`, `<nav>`, `<main>`, `<article>`, `<aside>`, `<footer>`
- Flexbox layout (no float)
- Profile link in navigation
- Image displayed inside `<aside>`
- Separate CSS file
- Responsive design implementation
